# AI-Chatbot-Testing-Framework

**PROJECT OVERVIEW-**
•	Developed an AI Chatbot Testing Framework using n8n and Gemini to automate validation of customer support responses.
•	Implemented intent matching and response quality checks to evaluate AI-generated answers across predefined test cases.
•	Automated result logging to Google Sheets with Pass/Fail status generation, enabling structured test reporting and faster defect identification.

 **FEATURES-**
 ✓ Automated test execution

✓ Intent validation

✓ Response quality checks

✓ Pass/Fail generation

✓ Google Sheets reporting

✓ Workflow automation
